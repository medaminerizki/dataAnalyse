Choisir un Type de Graphique

Line Plot

Visualise la variation et tendance d'une valeur en fonction d'une autre.
Axes : X : Quantitatives, Y : Quantitatives

Scatter Plot

Visualise les clusters et la corrélation (positive ou négative) entre deux variables.
Axes : X : Quantitatives, Y : Quantitatives

Boxplot

Résumé des statistiques d'une distribution autour de la médiane.
Axes : Un seul axe, valeurs quantitatives

Histogram

Affiche la distribution d'une variable numérique par rapport à une variable catégorielle.
Axes : X : Catégorique/Quantitatif, Y : Quantitatif

Violin Plot

Combiné avec un KDE plot, il permet de visualiser la distribution des données.
Axes : X : Catégorique, Y : Quantitatif

KDE Plot

Estimation de la densité de probabilité continue d'une variable.
Axes : X : Quantitatif, Y : Densité

Bar Plot

Compare des variables catégoriques par rapport à une variable numérique.
Axes : X : Catégorique, Y : Quantitatif

Heatmap

Visualise les relations entre plusieurs variables à l'aide de couleurs.
Axes : X : Catégorique/Quantitatif, Y : Catégorique/Quantitatif

Pie Chart

Utilisé pour afficher des proportions et des parts dans un ensemble.
Axes : Aucun axe spécifique, valeurs catégoriques.
